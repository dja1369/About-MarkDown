# About-MarkDown
## 1.    마크다운이란 무엇인가?
    마크다운이란 텍스트 기반의 마크업 언어로 2004 존그루버에 의해 만들어짐 
    쉽게 쓰고 읽을수 있으며 HTML로 변환가능
    특수기호와 간단한 구조의 문법을 이용하여 웹에서 보다 빠르게 작성하고 직관적인 생성 가능.
### 1.1 장점 
    1. 간결함
    2. 다양한 형태로 작성가능
    3. Text형태 이기에 용량이 적고 보관에 용이 
### 1.2 단점
    1. 표준이 없음
    2. 모든 HTML 마크업 대체 불가
